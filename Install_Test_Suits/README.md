# Install Test Suits
https://v1.quasar-framework.org/quasar-cli/testing-and-auditing  
##### Quasar Official Testing Page
https://testing.quasar-framework.org/  
### Install Test Suits
```
cd your-quasar-project
quasar ext add @quasar/testing
```
### Run tests
```
quasar test --unit jest
```
